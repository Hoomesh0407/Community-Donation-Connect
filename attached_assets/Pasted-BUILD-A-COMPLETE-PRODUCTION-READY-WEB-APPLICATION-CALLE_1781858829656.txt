BUILD A COMPLETE PRODUCTION-READY WEB APPLICATION CALLED "COMMUNITY DONATION CONNECT"

IMPORTANT INSTRUCTIONS

Generate a complete production-ready full-stack web application.

The application must be designed for real-world deployment and should include frontend, backend, database schema, APIs, authentication, multilingual support, maps integration, matching engine, trust system, admin dashboard, notifications, deployment configuration, GitHub integration support, and Vercel deployment readiness.

The final output must contain clean architecture, scalable code, responsive design, and proper documentation.

PROJECT OBJECTIVE

Community Donation Connect is a multilingual platform that helps donors and receivers connect directly within local communities.

The platform should allow users to donate useful items and allow needy people to request those items.

The system must intelligently match nearby donors and receivers based on:

* Item Category
* Item Type
* Distance
* User Verification
* Trust Score
* Receiver Urgency

The goal is to reduce waste and help communities support each other directly.

SMART MATCHING ENGINE

Core Feature

Automatically match donors and receivers.

Matching Rules

1. Same Category
2. Same Item Type
3. Match within selected radius
4. Verified Users First
5. Higher Trust Score First
6. Higher Urgency Receivers First

Distance Limit

Variable

Minimum:
1 km

Maximum:
10 km

The system must match based on whichever radius is selected by donor or receiver.

PRIVACY PROTECTION

IMPORTANT

Phone numbers and email addresses must NOT be visible before matching.

Before Match:

Hide:

* Phone Number
* Email
